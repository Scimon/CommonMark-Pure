NAME
====

CommonMark::Pure - Non compiled Implementation of CommonMark spec.

SYNOPSIS
========

```perl6
use CommonMark::Pure;
```

DESCRIPTION
===========

CommonMark::Pure is a Pure Raku Grammar based Markdown parser.
